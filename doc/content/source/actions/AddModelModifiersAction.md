# AddModelModifiersAction

## Overview

The `AddModelModifiersAction` allows you to modify an OpenMC model prior to running
a simulation with Cardinal. Examples of modifications that can be made includes:

- [OpenMCCellMaterialFill](OpenMCCellMaterialFill.md), to change a cell's material fill

!syntax list /Problem/ModelModifiers actions=false subsystems=false heading=Available ModelModifiers Objects

!syntax parameters /Problem/ModelModifiers/AddModelModifiersAction
